# MSI Z170A SLI PLUS
OpenCore hackintosh for msi Z170A SLI PLUS without using additional hardware
![Specs](/image.png)

__Specs__
- i7 6700K
- intel iGPU
- sata SSD

__Tested and working__

- [x] Intel HD Graphics 530
- [x] HDMI + HDMI Audio
- [x] Audio
- [x] Wired networking
- [x] iServices

__Not working__

- [ ] sleep (for now)
- [ ] wifi and bluetooth (now present)
